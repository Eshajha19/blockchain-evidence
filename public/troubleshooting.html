Troubleshooting

Common Issues and Solutions

üî¥ Problem 1: MetaMask Not Connecting
Symptoms: "Connect MetaMask" button doesn't work or shows error
Solutions:
  1. Ensure MetaMask extension is installed and enabled
  2. Check browser extensions (Settings > Extensions)
  3. Click MetaMask icon in toolbar - it should open
  4. Try refreshing the page (F5 or Cmd+R)
  5. Try disconnecting and reconnecting the wallet
  6. Clear browser cache and cookies
  7. Try a different browser (Chrome, Firefox, Safari, Edge)
  8. Restart your browser completely
üî¥ Problem 2: "Failed to Load Dashboard" Error
Symptoms: Error message on login, then dashboard loads
Solutions:
  1. This is usually temporary - wait a moment
  2. Click "OK" on error and continue
  3. If persists, refresh the page
  4. Clear browser cache (Ctrl+Shift+Delete)
  5. Try incognito/private browsing mode
  6. Check internet connection speed
  7. Try logging out and back in
üî¥ Problem 3: Evidence Upload Failing
Symptoms: Upload button doesn't work or upload fails
Solutions:
  1. Check file size (maximum 100MB)
  2. Verify file type is supported (images, videos, documents, audio)
  3. Check internet connection speed
  4. Try uploading smaller file first
  5. Try different browser
  6. Disable ad blockers and extensions
  7. Check available storage on device
  8. Make sure case is created before uploading
üî¥ Problem 4: Cases Not Showing in Dashboard
Symptoms: No cases visible even though cases exist
Solutions:
  1. Make sure case status is "Public" or you have permission
  2. Check your user role has case access
  3. Refresh the page (F5)
  4. Clear browser cache
  5. Try different browser
  6. Check if cases are for your department/jurisdiction
  7. Log out and back in
üü† Problem 5: Analysis Tools Not Working
Symptoms: Tools don't run or show error
Solutions:
  1. Upload evidence first - tools need data to analyze
  2. Check browser console for errors (F12 > Console)
  3. Reload page and try again
  4. Check if your role has analysis permissions
  5. Try different browser
  6. Disable browser extensions
  7. Check internet connection
üü† Problem 6: Wallet Address Not Recognized
Symptoms: System says wallet is not registered or asks to register
Solutions:
  1. Ensure connected to correct blockchain network
  2. Disconnect and reconnect wallet
  3. Check wallet address is spelled correctly
  4. Try different browser
  5. Clear browser cache
  6. Contact administrator to verify wallet is registered
  7. Make sure you have the correct MetaMask account selected
üü† Problem 7: Can't Access Admin Features
Symptoms: Admin buttons/pages don't appear
Solutions:
  1. Confirm your user role is "Administrator" (role 8)
  2. Contact system administrator to verify permissions
  3. Check if logged in properly
  4. Try logging out and back in
  5. Clear browser cache
  6. Try different browser
üü° Problem 8: Can't Download Evidence
Symptoms: Download button doesn't work or download incomplete
Solutions:
  1. Check browser download settings
  2. Check file permissions
  3. Try different browser
  4. Check available disk space
  5. Disable ad blockers
  6. Try downloading from different location
  7. Refresh page and try again
üü° Problem 9: Search Not Finding Cases
Symptoms: Search returns no results
Solutions:
  1. Try different search keywords
  2. Make sure cases you're searching for exist
  3. Check case status is "Public" or you have access
  4. Search for case number instead of title
  5. Refresh page and try again
  6. Clear search and try again
üü° Problem 10: PDF Report Generation Failing
Symptoms: Report download doesn't work or times out
Solutions:
  1. Close other browser tabs to free up memory
  2. Try with different browser
  3. Check internet connection speed
  4. Clear browser cache
  5. Try generating smaller report first
  6. Wait longer for generation to complete
  7. Contact support with error details
üü° Problem 11: Login Not Working
Symptoms: Can't login with email/password
Solutions:
  1. Check email address is correct
  2. Verify password is correct (case-sensitive)
  3. Try resetting password (forgot password link)
  4. Clear browser cache
  5. Try different browser
  6. Try incognito mode
  7. Disable VPN if using one
  8. Contact administrator if account locked
üü° Problem 12: File Upload Taking Too Long
Symptoms: Upload progress stuck or very slow
Solutions:
  1. Check internet connection speed
  2. Try uploading smaller file
  3. Make sure upload actually completed (check status)
  4. Reload page and check if file was uploaded
  5. Try different file format
  6. Close other applications using internet
  7. Try uploading again
